## 实现方案

### 1. 修改类型定义 (src/types/config.ts)

添加子菜单支持：
```typescript
export type NavBarLink = {
  name: string;
  url: string;
  external?: boolean;
  children?: NavBarLink[];  // 添加子菜单
};
```

### 2. 修改 Navbar 组件 (src/components/Navbar.astro)

- 添加多级菜单的 HTML 结构
- 添加鼠标悬停/点击展开子菜单的交互
- 使用 CSS 控制子菜单显示/隐藏

### 3. 修改 NavMenuPanel (src/components/widget/NavMenuPanel.astro)

- 支持移动端多级菜单展示
- 添加展开/折叠动画

### 4. 修改配置示例 (src/config.ts)

展示多级导航的使用方式：
```typescript
{
  name: "分类",
  url: "#",
  children: [
    { name: "技术", url: "/category/tech" },
    { name: "生活", url: "/category/life" },
  ]
}
```
