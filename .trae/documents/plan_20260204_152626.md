## 问题分析
点击评论按钮时页面跳动到文章前面，这是 Twikoo 的常见问题，通常是因为：
1. Twikoo 初始化时的 `jumpToComment` 行为导致页面滚动
2. 容器元素定位问题
3. 评论框获取焦点时浏览器默认行为

## 修复方案

### 1. 修改 Comment.astro 组件
**修复页面跳动问题**：
- 添加 `jumpToComment: false` 配置，防止 Twikoo 自动滚动到评论区
- 确保容器有正确的定位和 ID

**性能优化**：
- 恢复 `lazy: true` 懒加载模式，延迟加载评论系统
- 移除脚本预加载，让评论按需加载
- 使用更高效的加载策略

### 2. 改进加载策略
```javascript
twikoo.init({
  envId: 'https://plbt.hhj520.top/',
  el: '#tcomment',
  path: location.pathname,
  lang: 'zh-CN',
  lazy: true,
  jumpToComment: false,
  onCommentLoaded: function() {
    // 评论加载完成后的回调
  }
});
```

### 3. 测试验证
- 访问文章页面
- 点击评论相关按钮，确认页面不再跳动
- 验证评论功能仍然正常工作
