## 问题分析与修复方案

### 1. 腾讯云 EO 缓存配置
**问题**：需要确认和优化腾讯云 EO 的缓存策略
**修复**：
- 添加缓存控制头
- 配置静态资源长期缓存
- HTML 文件短缓存

### 2. 评论暗黑模式显示不清
**问题**：Twikoo 在暗黑模式下文字对比度不够
**修复**：
- 添加 Twikoo 暗黑模式 CSS 变量
- 调整文字颜色和背景色
- 确保在 `html.dark` 下的样式正确

### 3. 搜索速度慢
**问题**：Pagefind 搜索响应慢
**修复**：
- 预加载搜索索引
- 优化搜索初始化时机
- 添加搜索缓存

### 4. 文章链接显示错误
**问题**：License 组件显示 `https://fuwari.vercel.app` 而不是当前域名
**修复**：
- 修改 License.astro 使用相对路径或当前域名
- 修改 astro.config.mjs 的 site 配置
- 确保所有链接使用当前访问地址

## 具体修改文件
1. `vercel.json` - 添加缓存头配置（腾讯云 EO 兼容）
2. `src/components/Comment.astro` - 添加暗黑模式样式
3. `src/components/Search.svelte` - 优化搜索性能
4. `src/components/misc/License.astro` - 修复链接地址
5. `astro.config.mjs` - 更新 site 配置
