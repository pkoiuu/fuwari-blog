## 优化计划

### 1. JavaScript 性能优化

#### Swup 页面过渡优化
- 减少不必要的 hooks 监听
- 优化页面切换动画性能
- 禁用不常用的插件

#### 搜索功能优化
- Pagefind 延迟初始化
- 预加载搜索索引
- 优化搜索响应速度

### 2. CSS 优化

#### Tailwind CSS
- 精确配置 `content` 路径
- 移除未使用的样式

#### 关键 CSS 内联
- 首屏样式内联
- 异步加载非关键 CSS

### 3. 字体优化

#### 字体加载策略
- 添加 `font-display: swap`
- 预加载关键字体
- 系统字体 fallback

### 4. 构建配置优化

#### 代码分割
- 优化 manualChunks 配置
- 更细粒度的分割

#### 压缩优化
- Brotli/Gzip 压缩
- Terser 配置优化

### 5. 架构优化

#### 内容集合
- spec 集合添加 schema 验证
- 优化自动生成页面

#### i18n 优化
- 按需加载翻译文件
- 减少语言包大小

### 6. 构建分析

#### Bundle 分析
- 分析包大小
- 依赖优化

## 预期效果
- 页面加载速度提升 20-30%
- 页面切换更流畅
- 构建输出更小
- 用户体验提升
